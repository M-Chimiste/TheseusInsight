# React Frontend for Theseus Insight

This directory contains a minimal React application that can communicate with the existing FastAPI backend via REST and WebSocket endpoints.

## Available Scripts

- `npm run dev` – start the development server
- `npm run build` – build the app for production
- `npm run preview` – preview the production build

## Getting Started

Install dependencies with `npm install` and then run `npm run dev`.

The app expects the FastAPI backend to be running at the same host/port. Update the API URLs in the React code if needed.

This project uses a simple Vite configuration found in `vite.config.js`. Ensure you have Node installed and run `npm install` before starting `npm run dev`.
